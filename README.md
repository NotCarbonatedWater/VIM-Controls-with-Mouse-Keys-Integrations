# VIM Controls with Mouse Keys Integrations for Karabiner Elements
- "Shift"+"CapsLock" turns vim mode on/off
- tap "CapsLock" for normal mode 
- "h" = left arrow
- "j" = dowm arrow 
- "k" = up arrow
- "l" = right arrow
- "i" = insert mode (turns vim mode off for now)
- "m" activates mouse keys (control with "hjkl")
- "spacebar" for left click
- "semicolon" for right click
